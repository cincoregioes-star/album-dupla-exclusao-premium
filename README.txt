Como testar:
1. Abra index.html dentro desta pasta.
2. Clique em Abrir Pacote.
3. Cole figurinhas no álbum.
4. Responda simulados.
5. Teste o game.
6. Ao completar 36 figurinhas, registre a conclusão.

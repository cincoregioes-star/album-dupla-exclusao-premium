Para ativar Supabase:
1. Crie o projeto no Supabase.
2. Rode supabase/SUPABASE_COMPLETO.sql.
3. Edite supabase-config.js.
4. Troque enabled para true.
5. Cole Project URL e anon public key.
6. Teste a conclusão do álbum.

O álbum funciona mesmo sem Supabase, salvando os concluintes no navegador.
